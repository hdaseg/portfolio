Extraescolares

Sobre el proyecto

Este sistema web fue desarrollado como parte de mi proyecto integrador en la universidad, diseñado para centralizar la información de los estudiantes y mejorar la comunicación con los docentes. La plataforma incluye funciones para el registro de alumnos en actividades extraescolares y cuenta con módulos específicos para administradores, profesores y estudiantes. El objetivo principal fue crear una herramienta que permitiera gestionar datos, organizar actividades y consultar el progreso académico desde un solo lugar.

Tecnologías Usadas

HTML

Se utilizó para construir la estructura de los diferentes paneles de control (alumnos, docentes y administradores), asegurando que los formularios de registro y las listas de actividades estuvieran organizados.

CSS

Se empleó para el diseño integral de la interfaz, definiendo la apariencia de los módulos y la disposición de los elementos. Se trabajó en la jerarquía visual para que la información académica y los menús de navegación fueran claros y fáciles de usar.

JavaScript

Se utilizó para manejar la interactividad de la plataforma, incluyendo la validación de formularios y la implementación de un carousel para la visualización dinámica de contenido. También se encargó de la gestión de eventos para mejorar la experiencia del usuario en los módulos de consulta.

PHP

Fue el lenguaje encargado de la lógica del servidor, gestionando los diferentes niveles de acceso (roles de usuario) y la comunicación con la base de datos para guardar, modificar y mostrar el progreso de los alumnos.

PhpMyAdmin (SQL)

Se utilizó como sistema de gestión de base de datos para almacenar toda la información del proyecto. A través de esta herramienta se administraron las tablas de estudiantes, docentes y actividades, asegurando la integridad y persistencia de los datos escolares.
