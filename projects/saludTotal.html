Salud Total

Sobre el proyecto

Este sistema web fue desarrollado como un trabajo académico para la gestión de inventarios en una farmacia. El proyecto se centra en centralizar los datos de los medicamentos, permitiendo realizar las operaciones básicas de registro, consulta, actualización y eliminación (CRUD). El sistema facilita el control de existencias y la administración de proveedores, optimizando los procesos de entrada y salida de productos a través de una interfaz sencilla.

Tecnologías Usadas

HTML

Se utilizó para estructurar de manera semántica y sólida cada uno de los módulos del sistema (Inventario, Inicio de sesión, etc). El enfoque principal fue garantizar una jerarquía de información clara, permitiendo que los formularios de registro y las tablas de consulta fueran accesibles y fáciles de navegar.

CSS

Se empleó para la personalización detallada de la interfaz, permitiendo ajustar estilos específicos que refuerzan la identidad visual de la farmacia. Se trabajó en la legibilidad de tipografías y el diseño de componentes visuales únicos.

JavaScript

Aportó la interactividad necesaria para que la gestión fuera fluida y dinámica. Se implementó lógica en el lado del cliente para la validación de formularios y para confirmar acciones importantes como la eliminación de registros.

PHP

Actúa como el motor del lado del servidor, gestionando la lógica de negocio y la comunicación con la base de datos. Se utilizó para procesar las operaciones CRUD (Crear, Leer, Actualizar y Eliminar), administrar las sesiones de usuario y asegurar que el control de existencias se actualice de forma precisa con cada entrada y salida del inventario.

Bootstrap

Se utilizó como el desarrollo principal de diseño para garantizar un desarrollo ágil y una estructura totalmente responsiva. Gracias a su sistema de rejilla y componentes predefinidos como tarjetas, botones y barras de navegación, se logró una interfaz profesional, estandarizada y adaptable a cualquier tamaño de pantalla.
