HealthTrack+

Sobre el proyecto

HealthTrack es una aplicación móvil desarrollada como proyecto académico para el seguimiento de indicadores de salud personales. La aplicación permite a los usuarios registrar y monitorear datos como la actividad física, el consumo de agua y otros parámetros de bienestar. El objetivo principal fue reunir esta información en un dispositivo móvil para facilitar el control personal y ofrecer una visualización clara del progreso diario a través de una interfaz sencilla.

Tecnologías Usadas

Kotlin

Se utilizó para desarrollar la lógica principal de la aplicación, gestionando el procesamiento de los datos de salud y la navegación entre las diferentes pantallas de la interfaz.

XML

Se empleó para definir la estructura visual de la aplicación, asegurando que los elementos de entrada de datos y las gráficas de seguimiento fueran claros y fáciles de usar.

Firebase (SQL)

Se implementó una base de datos online para el almacenamiento de la información. Esto permite que los datos se sincronicen en tiempo real, logrando que el usuario pueda acceder a su misma información desde diferentes dispositivos de manera simultánea.

Figma

Se utilizó para el diseño previo de las interfaces y el prototipado del flujo de navegación. Esta herramienta permitió definir la experiencia de usuario (UX) y el aspecto visual (UI) antes de pasar a la etapa de desarrollo.
